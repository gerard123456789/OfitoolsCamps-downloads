# Ofitools Camps Downloads

This repository is the public download site for Ofitools Camps.

It is intended for:

- public release assets
- GitHub Pages download links
- release notes and version visibility

This repository does not document the private source code or internal implementation details of the application.

## About Ofitools Camps

Ofitools Camps is a practical office utility focused on everyday document tasks, including:

- PDF merge and page editing
- PDF compression
- spreadsheet comparison with highlighted differences

Public downloads are provided for:

- Windows
- Android
- Mac

## Downloads

The latest public version is published through GitHub Releases in this repository.

Typical release assets include:

- `OfitoolsCamps-win-x64.zip`
- Android `.apk`
- `OfitoolsCamps-maccatalyst.zip`

The GitHub Pages homepage in this repository links directly to the latest public release assets.
